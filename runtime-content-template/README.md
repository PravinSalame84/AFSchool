# Runtime Content Sheet Template

These CSV files match the Google Sheet tabs required by the runtime content system.

## How To Use

1. Create a new Google Sheet.
2. Create tabs with these exact names:
   - `site_status`
   - `announcement_actions`
   - `marquee`
   - `notices`
   - `events`
   - `downloads`
3. Open each CSV file in this folder.
4. Copy the header row and sample rows into the matching sheet tab.
5. Replace sample content with your school's live content.
